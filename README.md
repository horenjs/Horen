# Horen

Horen 是一个用 React 和 electron 打造的本地音乐播放器，正在不断开发中

预览图：

![preview-main](./preview.jpg)

功能进度

- [x] 主体界面
- [x] 当前播放
- [x] 播放队列
- [x] 控制播放进度
- [ ] 控制音量
- [ ] 歌词显示
- [ ] 调音器
- [ ] 播放主界面



## 开发

```bash
git clone https://github.com/kevinjobs/Horen.git

cd Horen

# if you don't install pnpm
npm i -g pnpm

# then
pnpm install

# start the app
pnpm start --filter horen
```
