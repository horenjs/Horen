## 功能进度

- [x] 主体界面
- [x] 当前播放
- [x] 播放队列
- [x] 控制播放进度
- [x] 设置界面
- [ ] 控制音量
- [ ] 歌词显示
- [ ] 调音器
- [ ] 播放主界面



## 开发

```bash
git clone https://github.com/kevinjobs/Horen.git

cd Horen

# if you don't install pnpm
npm i -g pnpm

# then
pnpm install

# start the app
pnpm start --filter horen
```
